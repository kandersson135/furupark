FURUPARKSKOLAN

SPÖKEN PÅ RYMMEN

Välj spelare:


Hur du spelar:

Spela som rektor Fredrik eller
skolhandläggare Renée och fånga spöken.
Om spöket hinner försvinna innan du
fångar det, tappar du ett liv.
Försök att fånga så många spöken som
möjligt, men se upp, spökena dyker upp
på de mest luriga platserna.

Använd WASD för att flytta spelaren.
Tryck "Enter" för att fånga ett spöke.
